# 03_day

BIOS中断号`0x13`:

简单理解就是对磁盘操作，通过`AH`的取值不同来进行不同的中断操作，简单例子

	AH=02H 读扇区
	AH=03H 写扇区
	AH=04H 校验扇区

上述操作只是一部分

其他寄存器设置参数

	AL=扇区数（必须为连续扇区）
	CH=柱面
	CL=扇区（0-5位）|（柱面号&0x300）>> 2
	DH=磁头号
	DL=驱动器号
	ES:BX=要写入的缓冲区
	调用成功CF=0,AH=0
	调用失败CF=1,AH=error code
	
一张软盘有两个磁头，80个柱面，18个扇区，所以一个软盘的大小为`2x80x18x512`个字节（当然这都是二十年前的东西了）
