`pushfd`，`popfd`指令：

由于标志位中的值不能直接赋值到通用寄存器中，所以采用栈做中间变量做赋值

`OUT`指令，向设备发送电信号

`IN`指令，从设备取电信号

`port`参数为识别各不同的设备
